## 🚀 Overview

This portfolio is built to reflect my passion for Linux, systems, and automation. It's not just a website; it's a functional terminal environment where you can explore my work, skills, and labs through interactive commands.

### ✨ Key Features

- **Full Terminal Interface** — A custom-built terminal emulator with 20+ commands.
- **Project Labs** — Interactive showcase of systems and DevOps projects.
- **SEO Optimized** — Machine-readable `llms.txt` and `profile.json` for AI agents.
- **Mobile Responsive** — Seamless terminal experience across all devices.

---

## 🛠️ Commands to Try

- `who` — Learn about my background and interests.
- `projects` — Explore my systems and DevOps work.
- `skills` — View my technical stack.
- `neofetch` — See your current system info.
- `help` — List all available commands.

---

## 💻 Tech Stack

- **Frontend**: React 19, Vanilla CSS
- **Data**: JSON-driven architecture for projects and profile
- **Metadata**: Automated sync for LLM-friendly documentation
- **Deployment**: GitHub Pages

---

## 🛠️ Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vedantdubal-141/vedantdubal-141.github.io.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
